# ble_test

Flutter project: 可重複測量BLE裝置的訊號強弱(RSSI)

使用套件：[flutter_blue](https://pub.dev/packages/flutter_blue)

## 注意事項
- minSdkVersion版本可能衝突
    目前改法是將 build.gradle 的 minSdkVersion 更改為合適版本
    
- scanResults的結果可能無法重新，需更改原始碼
